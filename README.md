# 🧮 BMI Calculator

A simple GUI-based Body Mass Index (BMI) Calculator built using Tkinter.

## Features

- Calculate BMI using weight and height
- Input validation
- BMI category classification:
  - Underweight
  - Normal Weight
  - Overweight
  - Obese
- User-friendly graphical interface

## Formula

BMI = Weight (kg) / Height² (m²)

# 🎙️ Simple Voice Assistant

A beginner-friendly voice assistant that uses text input and speech output.

## Features

- Responds to greetings
- Answers simple predefined questions
- Basic search query handling
- Text-to-speech output
- Exit command support

## Supported Commands

| Command | Response |
|----------|-----------|
| Hello | Greeting |
| How are you | Status response |
| Your name | Assistant introduction |
| Search Python | Search response |
| Bye / Exit | Close assistant |

## Technologies Used

- pyttsx3
